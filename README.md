# 股票分析系统

## 项目简介

用于股票数据获取、指标计算、策略分析、回测评估与可视化展示的综合分析系统，辅助进行量化研究与投资决策分析。

## 核心功能

- 股票行情数据采集
- 技术指标计算
- 策略回测引擎
- 风险收益分析
- 可视化仪表盘
- 数据存储与任务调度

## 技术栈规划

- Python
- Pandas / NumPy
- FastAPI
- Plotly / ECharts
- SQLite / PostgreSQL
- Backtrader / 自研回测框架
